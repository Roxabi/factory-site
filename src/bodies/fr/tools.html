Roxabi Factory · Outils

Outils disponibles

La couche outil est ce par quoi les agents Factory agissent sur le monde — voix, image, artefacts visuels, vidéo, curation, publication sociale. Chaque outil est une surface de capacité adossée à un provider ; les workers consomment des outils mais n'en sont pas. Les CLIs listés ici fonctionnent de manière autonome aujourd'hui. Leur intégration dans le moteur Factory comme providers first-class de la couche tool est sur la feuille de route, suivie sous #1670.

Cette page recense les instances de la couche tool. Pour le modèle — taxonomie, vocabulaire runtime, distinction domaine-nature — voir Architecture des outils.

Primitives Roxabi

CLIs de capacité auto-hébergés

Outils open source construits et maintenus par Roxabi. Chacun fonctionne de façon autonome sur votre propre matériel — aucun lock-in cloud, aucune télémétrie. Lorsque la couche tool sera disponible, ils s'enregistreront comme providers satellites sur le bus NATS de Factory.

voiceCLI

En service

Voix · TTS / STT

Un seul CLI pour la synthèse vocale et la transcription en local. Pilote Qwen3-TTS, Chatterbox et Whisper depuis un script Markdown unique. Un satellite NATS permet de diffuser la voix sur tous les nœuds du réseau.

voiceCLI sur roxabi.dev →

imageCLI

À la demande

Image · génération

Un seul CLI pour la génération d'images en local. FLUX.2, FLUX.1 (+ verrou de visage PuLID) et SD3.5 derrière une seule commande. Quantisation et gestion VRAM automatiques. Satellite NATS pour le rendu cross-host. Activé lorsqu'un nœud GPU est en ligne.

imageCLI sur roxabi.dev →

roxabi-forge

En service

Artefacts visuels · diagrammes, galeries, slides

Un plugin Claude Code qui transforme « explique-moi ça » en artefact HTML autonome — diagrammes, galeries, présentations, guides. Huit skills, déploiement en une commande. Candidat à un domaine tool d'artefacts visuels dans la couche Factory.

roxabi-forge sur roxabi.dev →

roxabi-production

À la demande

Vidéo · génération

Pipeline de génération vidéo. Combine génération d'images, synthèse audio et composition de scènes pour produire des vidéos finies. S'exécute à la demande sur du matériel GPU. Candidat à un domaine tool vidéo.

roxabi-intel

À la demande

Curation · lien → contenu structuré

Transforme un flux de liens Discord en documents Markdown structurés et en galeries indexées. Planifié ou déclenché, sans pipeline manuel. Correspond au futur domaine tool scrape dans la taxonomie Factory.

Intégration externe

Surface de publication tierce

La couche publication sociale relie les jobs Factory aux plateformes externes. Un adaptateur satellite NATS fera le pont entre le backing service et la couche tool.

Postiz

Bientôt

Publication sociale · jobs CM

Publication sociale auto-hébergée via Postiz. Un backing service roxabi-postiz (web + DB + Redis) est exposé par un adaptateur satellite NATS qui crée un domaine tool postiz. L'adaptateur est suivi sous #1713. Les jobs CM de Factory dispatacheront les actions de publication via cet outil dès que l'adaptateur sera disponible.
